# PayNexus 💳

PayNexus is an ultra-premium, mobile-first payment management platform designed with a sophisticated, Apple-inspired aesthetic. It provides a seamless experience for tracking analytics, managing groups, and processing payments with a focus on high-end UI/UX.

![PayNexus Banner](https://images.unsplash.com/photo-1563013544-824ae1b704d3?auto=format&fit=crop&q=80&w=2070)

## ✨ Features

- **Ultra-Premium Design**: A "machined" glassmorphism aesthetic with metallic typography and smooth micro-interactions.
- **Mobile-First Experience**: Native-app feel with a fixed bottom navigation bar and sliding bottom-sheet modals.
- **Comprehensive Dashboard**: Real-time analytics and transaction tracking.
- **Group Payments**: Easily manage and split payments within groups.
- **Secure Authentication**: Robust sign-in/up flow with password strength feedback and secure reset functionality.
- **Responsive Layouts**: Perfectly optimized for both mobile and desktop viewing.

## 🛠️ Technology Stack

- **Frontend**: Semantic HTML5, Modern Vanilla CSS (Custom Design System), Vanilla JavaScript.
- **Backend/Integrations**: Node.js, Nodemailer (for automated notifications).
- **Design Philosophy**: Minimalist, Apple-inspired "Machined Glass" UI.

## 🚀 Getting Started

### Prerequisites

- A modern web browser.
- [Node.js](https://nodejs.org/) (for backend services like email notifications).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/CodeCrafter3-0-2/PayNexus-CC3H-116.git
   cd PayNexus
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the app**:
   Open `index.html` in your browser or use a local dev server (e.g., Live Server).

## 📱 Mobile Interface

The app is specifically optimized for mobile devices, featuring:
- **Bottom Navigation**: Quick access to Home, Analytics, Groups, and Settings.
- **Action Center**: Centered "Pay" button for immediate transaction access.
- **Safe Area Support**: Full compatibility with modern smartphone displays.
